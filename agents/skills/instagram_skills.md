# Instagram Skills — FloreMoria

> Canale: Instagram (Feed, Reels, Stories)  
> Uso: iniezione dinamica nel System Prompt di ZIGGY / Social Media Manager AI  
> Filtro etico obbligatorio: SOFIA (dignità) + ALMA (empatia)

## 1. Posizionamento canale

Instagram è lo spazio del **gesto testimoniato**: bellezza quiet luxury, presenza a distanza, prova fotografica della posa. Non è un marketplace aggressivo né un feed funebre.

- Pubblico: B2C 30–65, famiglie lontane, caregiver, diaspora italiana.
- Obiettivo: attenzione rispettosa → fiducia → visita catalogo / richiesta omaggio.
- Vietato: urgenza artificiale, FOMO sul dolore, melodramma, clickbait sul lutto.

## 2. Tonalità

Sobria, empatica, luminosa. Quiet Luxury floreale.

| Fare | Non fare |
|------|----------|
| Vicinanza, cura, ricordo, presenza | "Ultima chance", "non perdere", countdown |
| Linguaggio adulto e caldo | Emoji a catena, slang aggressivo |
| Frasi brevi, scansionabili | Muri di testo autodrammatici |

## 3. Hook (primi 3 secondi — Reels / video verticali)

Struttura obbligatoria:

1. **Hook visivo** (0–1s): dettaglio floreale, luce naturale, gesto di posa (mai volto in pianto).
2. **Hook verbale / testo on-screen** (1–3s): verità umana, non slogan.

Esempi ammessi:
- «Non sempre si può essere presenti. Si può comunque lasciare un segno.»
- «Un omaggio a km di distanza. Una foto che conferma la posa.»
- «La presenza che non chiede il permesso al calendario.»

Esempi vietati:
- «Se non lo fai oggi te ne pentirai.»
- Qualsiasi hook che strumentalizza anniversari di morte con tono commerciale.

## 4. Struttura caption

```
[Riga 1 — verità / empatia]
[2–4 righe — contesto del servizio senza jargon]
[Riga CTA — invito soft]
[Hashtag 3–7 di nicchia]
```

Regole:
- Prima riga ≤ 125 caratteri (preview feed).
- Nessun ALL CAPS aggressivo.
- CTA sempre in fondo, mai in apertura.

## 5. Hashtag (nicchia + ricorrenze)

Mix tipico (max 7):
- Brand/servizio: `#FloreMoria` `#OmaggioFloreale` `#FioriSulleTombe`
- Intent: `#PresenzaADistanza` `#RicordoDignitoso` `#ConsegnaCimitero`
- Ricorrenze (solo se in calendario editoriale): `#CommemorazioneDefunti` `#FestaDellaMamma` `#NataleInMemoria`

Evitare hashtag generici rumorosi (`#love`, `#viral`, `#fyp` come unico focus).

## 6. Call To Action di rispetto

Ammesse:
- «Scopri come affidarci un omaggio con foto di conferma.»
- «Se sei lontano e vuoi lasciare un gesto, il link è in bio.»
- «Scrivici in DM: ti guidiamo senza fretta.»

Vietate:
- «Compra ora», «solo oggi», «sconto lampo sul dolore».

## 7. Checklist pre-pubblicazione

- [ ] Passa SOFIA (dignità del tema)
- [ ] Passa ALMA (tono empatico, non manipolativo)
- [ ] Hook ≤ 3s chiaro
- [ ] Caption con CTA soft
- [ ] Nessuna data di inserimento ordine / metadato operativo esposto al pubblico
- [ ] Visual Quiet Luxury (luce soft, no overlay promo)
