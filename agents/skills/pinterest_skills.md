# Pinterest Skills — FloreMoria

> **Canale:** Pinterest (Pin statici, Idea Pin / video verticali dove utile)  
> **Uso:** iniezione runtime nel System Prompt (ZIGGY / Social Media Manager AI)  
> **Filtro etico obbligatorio:** SOFIA (dignità) + ALMA (empatia)  
> **Principio guida:** ispirazione e intenzione di ricerca — mai speculazione sul dolore.

---

## 1. Target e tono di voce

### 1.1 Target primario

| Segmento | Bisogno | Messaggio chiave |
|----------|---------|------------------|
| Planner / famiglie in ricerca | Idee per omaggio dignitoso | «Ispirazione Quiet Luxury per la tomba» |
| Utenti stagionali / ricorrenze | Board per Defunti, anniversari, feste | «Salva per quando servirà un gesto» |
| Amanti di piante e allestimenti | Estetica botanica e composizione | Focus su **piante** e **allestimenti tombali** sobri |

**Mentalità Pinterest:** l’utente *salva* per dopo. Tono da guida visuale, non da chat social.

### 1.2 Tono di voce

- **Eleganza** editoriale: silenziosa, premium, ispirazionale.
- **Empatia** indiretta: il Pin aiuta a scegliere senza forzare l’emozione.
- **Rispetto del ricordo:** niente dramatizzazione; focus su bellezza e cura.
- **Mai speculativo:** vietati badge sconto, countdown, griefbait in overlay.

#### Lessico preferito

`omaggio floreale` · `allestimento sobrio` · `pianta in vaso` · `quiet luxury` · `posa dignitosa` · `ispirazione` · `presenza a distanza`

#### Lessico vietato

`offerta lampo` · `viral pin` · `shock` · linguaggio commerciale sul dolore · clickbait

#### Registro

Titoli keyword-first ma umani. Descrizioni utili. Meno “story chat”, più “catalogo ispirazionale con anima”.

## 2. Hook visivi e testuali dei primi 3 secondi

*Per Idea Pin / video: verticale. Per Pin statico: l’hook è il **crop 2:3** + testo overlay ≤ 8 parole.*

### 2.1 Video / Idea Pin (se usati)

| Tempo | Obiettivo | Esecuzione |
|-------|-----------|------------|
| **0–1 s** | Bellezza immediata | Still o slow pan su pianta / bouquet / allestimento |
| **1–3 s** | Promessa | Overlay: categoria o beneficio («Omaggio sobrio per la tomba») |
| **3–15 s** | Valore | Dettagli botanici, texture, contesto di posa autorizzato |
| **Close** | CTA soft | «Scopri su FloreMoria» / salva il Pin |

### 2.2 Hook visivi prioritari (focus FloreMoria)

- **Piante in vaso** commemorative (es. Kalanchoe e analoghe) in luce naturale.
- **Allestimenti tombali** sobri: composizione su pietra chiara, proporzioni equilibrate.
- Dettaglio foglia / petalo → wide sull’allestimento completo.
- Mood board Quiet Luxury: legno, lino, ceramica, verde salvia.

### 2.3 Hook testuali overlay (max 6–8 parole)

- «Omaggio sobrio per la tomba»
- «Presenza a distanza, con foto»
- «Pianta commemorativa Quiet Luxury»
- «Allestimento dignitoso, luce naturale»

### 2.4 Vietato

- Overlay promo, face-cam melodrammatica, foto di tombe altrui riconoscibili, watermark rumorosi.

## 3. Struttura delle caption (titolo + descrizione Pin)

### 3.1 Framework: Storytelling + Valore + CTA

**Titolo Pin** (SEO + empatia):

```text
[Keyword naturale] | [Beneficio / stile] | FloreMoria
```

**Descrizione:**

```text
[1 frase di storytelling / senso]
[Valore concreto: composizione, pianta, cosa include il servizio]
[CTA soft verso catalogo / landing]
[Hashtag 2–5 di nicchia]
```

### 3.2 Esempio (pianta + allestimento)

**Titolo:** `Pianta in vaso per omaggio tombale | Quiet Luxury | FloreMoria`

**Descrizione:**

```text
Quando si cerca un gesto che duri oltre il giorno della visita,
una pianta curata può accompagnare il ricordo con discrezione.

Allestimento sobrio, luce naturale, consegna sulla tomba
con foto di conferma della posa.

Salva questo Pin e scopri la composizione sul sito FloreMoria.

#FloreMoria #OmaggioFloreale #FioriSulleTombe #PianteCommemorative #QuietLuxury
```

### 3.3 CTA di impatto e rispettose

| Ammesse | Vietate |
|---------|---------|
| «Scopri la composizione sul sito FloreMoria.» | «Compra prima che finisca» |
| «Salva questo Pin per quando servirà un gesto dignitoso.» | Pressure selling |
| «Apri il link per dettagli sulla posa.» | Confronto aggressivo con concorrenti |

## 4. Strategia hashtag ed algoritmo

### 4.1 Keyword first, hashtag second

Su Pinterest l’algoritmo privilegia **titolo, descrizione e alt text**. Gli hashtag sono di supporto (2–5).

| Tipo | Esempi keyword nel titolo/descrizione | Hashtag di supporto |
|------|----------------------------------------|---------------------|
| Servizio | omaggio floreale, fiori sulle tombe, consegna cimitero | `#OmaggioFloreale` `#FioriSulleTombe` |
| Prodotto | pianta in vaso, bouquet commemorativo, allestimento tombale | `#PianteCommemorative` `#AllestimentoTomba` |
| Brand / stile | FloreMoria, quiet luxury, presenza a distanza | `#FloreMoria` `#QuietLuxury` `#PresenzaADistanza` |
| Ricorrenza | commemorazione defunti, festa della mamma in memoria | Solo se board/campagna attiva |

### 4.2 Geolocalizzazione

- Inserisci **città/regione** in descrizione se il Pin è market-specific («consegna cimiteri in Lombardia»).
- Usa board geo (es. “Omaggi — Nord Italia”) senza esporre luoghi sensibili.
- Evita pin su mappe di cimiteri con foto invasive.

### 4.3 Frequenza raccomandata

| Formato | Frequenza | Note algoritmo |
|---------|-----------|----------------|
| Pin statici **2:3** | 5–10 / settimana | Fresh pins > repin eccessivi; varia titoli |
| Idea Pin / video | 1–3 / settimana | Hook 3s + focus piante/allestimenti |
| Board curation | Continua | Board per intenzione + categoria + ricorrenza |

**Segnali:** salvataggi, close-up, outbound click, tempo sul Pin.  
**Freshness:** rinnova creative ogni stagione (Defunti, primavera, Natale in memoria).

## 5. Specifiche visive e layout

### 5.1 Formato master — Pin verticali **2:3**

| Specifica | Valore |
|-----------|--------|
| Ratio consigliato | **2:3** (es. 1000×1500 px) |
| Formato minimo utile | 600×900 |
| Idea Pin / video | 9:16 accettato; thumbnail coerente 2:3 |
| Testo overlay | ≤ 20% area; max 6–8 parole |

### 5.2 Focus creativo obbligatorio FloreMoria

1. **Piante** commemorative e in vaso (durata, cura, sobrietà).  
2. **Allestimenti tombali** eleganti: proporzioni, pietra, luce, niente clutter.  
3. Bouquet / composizioni Quiet Luxury come supporto, non come “promo fioreria”.

### 5.3 Layout Pin

```text
┌─────────────────────┐
│  (aria / luce)      │  ← terzo superiore: respiro
│                     │
│   SOGGETTO          │  ← centro: pianta o allestimento
│   (fuoco netto)     │
│                     │
│  [testo overlay]    │  ← basso-centro, tipografia soft
└─────────────────────┘
     ratio 2:3
```

### 5.4 Palette e stile (ARLO)

- Avorio, cipria, salvia, grigio pietra; ombre morbide.
- Nessun bordo promo, sticker, badge %, collage rumoroso.
- Alt text descrittivo e rispettoso (accessibilità + SEO).

### 5.5 Board strategy (sintesi)

| Board | Contenuto |
|-------|-----------|
| Piante commemorative | Focus vaso / durata / cura |
| Allestimenti tombali | Composizioni sobrie, riferimenti stile |
| Ricorrenze | Defunti, anniversari, feste in memoria |
| Presenza a distanza | Educativo + servizio + foto di conferma |

### 5.6 Checklist pre-pubblicazione

- [ ] Creative **2:3** leggibile in miniatura  
- [ ] Focus piante e/o allestimenti tombali  
- [ ] Titolo con keyword naturali  
- [ ] Descrizione = storytelling + valore + CTA soft  
- [ ] 2–5 hashtag di nicchia (+ geo testuale se pertinente)  
- [ ] SOFIA + ALMA · nessun dark pattern
