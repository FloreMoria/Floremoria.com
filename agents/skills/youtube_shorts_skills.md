# YouTube Shorts Skills — FloreMoria

> Canale: YouTube Shorts (verticale, discovery + evergreen)  
> Uso: iniezione dinamica nel System Prompt di ZIGGY / Social Media Manager AI  
> Filtro etico obbligatorio: SOFIA (dignità) + ALMA (empatia)

## 1. Posizionamento canale

Shorts è ideale per **spiegazioni evergreen** del servizio: come funziona FloreMoria, cosa include la foto di conferma, differenze rispetto a una fioreria tradizionale.

- Pubblico: ricerca intenzionale + discovery; età mista.
- Obiettivo: chiarezza → fiducia → click verso sito (descrizione / end screen soft).
- Vantaggio: titoli e descrizioni SEO-friendly (AXEL).

## 2. Tonalità

Educativa, calma, premium. Più “documentario breve” che trend.

## 3. Hook (primi 3 secondi)

Obbligatorio testo grande + enunciato chiaro:

- «FloreMoria in 20 secondi.»
- «Omaggio floreale sulla tomba: come funziona davvero.»
- «Perché la foto dopo la posa è parte del servizio.»

Framework:

```
0–3s   Promessa chiara (cosa imparerà lo spettatore)
3–20s  3 beat: problema umano → soluzione → prova (foto)
20–35s Riepilogo + CTA soft
```

## 4. Titolo e descrizione

**Titolo** (≤ 70 caratteri): beneficio + contesto, niente clickbait.
Esempi:
- «Omaggio floreale a distanza: come funziona FloreMoria»
- «Foto di conferma dopo la posa: perché è importante»

**Descrizione**:
```
[2–3 frasi di contesto empatico]
[Cosa include il servizio]
[CTA soft + URL]
[Hashtag 3–6]
```

## 5. Caption / testo on-screen

- Sottotitoli sempre (accessibilità).
- Una idea per Short: non mischiare 4 messaggi.
- Closing card statica con brand + frase di rispetto.

## 6. Hashtag e ricorrenze

- `#FloreMoria` `#OmaggioFloreale` `#YouTubeShorts`
- `#FioriSulleTombe` `#PresenzaADistanza`
- Ricorrenze in titolo solo se davvero editoriali (mai exploit del dolore)

## 7. Call To Action di rispetto

Ammesse:
- «Trovi il dettaglio del servizio nel link in descrizione.»
- «Se stai cercando un gesto dignitoso a distanza, inizia da lì.»
- «Iscriviti se vuoi altri chiarimenti sul servizio — senza rumore.»

Vietate: CTA aggressive, “limited time”, confronti denigratori verso fioristi locali.

## 8. Checklist

- [ ] Hook informativo ≤ 3s
- [ ] Titolo SEO sobrio
- [ ] Sottotitoli
- [ ] CTA soft in descrizione
- [ ] SOFIA + ALMA
