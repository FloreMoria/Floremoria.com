# TikTok Skills — FloreMoria

> Canale: TikTok (video verticali 9:16)  
> Uso: iniezione dinamica nel System Prompt di ZIGGY / Social Media Manager AI  
> Filtro etico obbligatorio: SOFIA (dignità) + ALMA (empatia)

## 1. Posizionamento canale

TikTok richiede **autenticità e ritmo**, senza mai scivolare in trash o trend che banalizzano il lutto.

- Pubblico: eterogeneo, spesso più giovane; UGC-feel accettato se dignitoso.
- Obiettivo: fermare lo scroll → spiegare il gesto → soft CTA verso sito / bio.
- Formato: verticale, montaggio netto, testo on-screen essenziale.

## 2. Tonalità

Umana, diretta, sobria. Colloquiale sì; cinica no. “Quiet luxury” anche in UGC: mani che compongono, luce naturale, niente effetti comici sul tema.

## 3. Hook obbligatorio (0–3 secondi)

Framework fisso:

```
0–1s  Hook visivo (dettaglio / gesto / testo grande)
1–3s  Hook verbale: domanda o verità (una sola)
3–12s Body: come funziona / perché conta
12–25s Close: CTA soft + brand
```

Hook ammessi:
- «Come porti un fiore a chi ami se sei a 800 km?»
- «Questa non è una fioreria. È una presenza delegata.»
- «Guarda cosa succede dopo la posa.»

Hook vietati:
- Sound virali ironici sul funerale.
- Shock griefbait, pianti forzati, “duet” su contenuti di lutto altrui senza consenso.

## 4. Caption TikTok

Breve (≤ 150 caratteri consigliati) + CTA soft + 3–5 hashtag.

```
[Una riga di senso]
[CTA soft]
#FloreMoria #OmaggioFloreale #PresenzaADistanza …
```

## 5. Hashtag di nicchia

- Sempre: `#FloreMoria`
- Nicchia: `#FioriSulleTombe` `#OmaggioFloreale` `#Ricordo`
- Discovery controllata: `#ConsigliUtili` `#CoseDaSapere` (mai `#fyp` come unica strategia)
- Ricorrenze: solo se allineate al calendario CLEO/MARK

## 6. Call To Action di rispetto

Ammesse (voce o testo):
- «Link in bio se vuoi saperne di più, senza fretta.»
- «Salva questo video se un giorno potrebbe servirti.»
- «Commenta “info” e ti indichiamo dove iniziare.»

Vietate: “compra subito”, giveaway legati al dolore, pressure selling.

## 7. Audio e testo on-screen

- Preferire voice-over calma o musica strumentale soft.
- Testo on-screen grande, poche parole, alto contrasto accessibile.
- Mai overlay promo aggressivi (badge sconto, sticker urgenza).

## 8. Checklist

- [ ] Hook ≤ 3s
- [ ] Nessun griefbait
- [ ] Close con CTA soft
- [ ] SOFIA + ALMA
- [ ] Brand riconoscibile senza gridare
